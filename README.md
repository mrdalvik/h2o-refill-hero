# H2O: Refill Hero

Пиксельная веб-игра для отслеживания потребления воды. Заполняй вендинговый автомат бутылочками каждый раз, когда пьёшь воду!

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Как играть

1. Нажми на панель ввода на автомате
2. Введи количество выпитой воды в миллилитрах
3. Нажми "Поставить на полку" — бутылочка появится в автомате
4. Достигни дневной цели в 2000 мл!

### Размеры бутылок

- До 200 мл — маленькая бутылочка
- 201-500 мл — средняя бутылка
- 501-1000 мл — большая бутылка
- Больше 1000 мл — несколько бутылок

### Смена дня

Каждый новый день к автомату подходят покупатели, забирают воду, и автомат снова пустеет.

**Dev-код:** введи `0000` и нажми "Поставить на полку" для сброса дня.

## Технологии

- Vue 3 + TypeScript + Composition API
- Pinia (state management)
- @pixelium/web-vue (pixel-art UI)
- Vite
- localStorage (persistence)
